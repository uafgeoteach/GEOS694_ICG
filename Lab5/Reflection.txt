#### Reflection
- What behavior do you observe in your scaling?
  Running on more cores speeds it up, so long as you have more cores to run it.

- Are there diminishing returns for scaling up?
  Yes

- What limitations are we running into when we throw more and more cores at it, going past the physical number of cores on our machine?
  Cores start needing to run things in series instead of in parallel, so we start making each core run these computations twice, three times, etc.  This begins to counteract the benefits of running in parallel, and the script takes longer to run to completion.
